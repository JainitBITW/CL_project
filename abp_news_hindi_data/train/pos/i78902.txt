MSN rebranding द्वारा प्रभावित सेवाओं में से कुछ जो Windows Live Hotmail बन गया MSN Hotmail, शामिल है; जो Windows Live मैसेन्जर हो गया MSN मैसेन्जर, जो ; Live Search बन गए एमएसएन खोज, जो जीते खोज Maps बन गए एमएसएन वर्चुअल पृथ्वी, जो Windows Live रिक्त स्थान बन गए एमएसएन Spaces, जो Windows Live चेतावनी बन गए एमएसएन चेतावनी और MSN समूह, जो Windows Live समूह बन गया है। कुछ अन्य संबंधित सेवाओं, जैसे MSN प्रत्यक्ष, के रूप में परिवर्तित करने के लिए Windows Live के बिना MSN परिवार का एक हिस्सा रहा है।
